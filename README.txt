Este pacote contém as telas do sistema de alocação da Claro:
- dashboard.html: tela de lançamento por calendário
- aprovacao.html: visualização por gestor, 12 meses em colunas com checkbox por mês
- relatorio.html: filtros com botão de inversão da ordem (tipo > projeto > HC etc.)
- login.html: tela de entrada
- style.css: estilo responsivo unificado
- js/: scripts das telas
- app.py: servidor Flask

Os arquivos JSON e a logo devem ser mantidos conforme já carregados anteriormente.
